### 快捷键

C --> Ctrl

S --> Shift

M --> Alt

| 功能         | 快捷键 |
|:-------------|:-------|
| 返回         | M-,    |
| 前进         | M-.    |
| 跳到定义     | C-=    |
| 查找引用     | C-/    |
| 搜索         | C-f    |
| 向下搜索     | F4     |
| 向上搜索     | F3     |
| 高亮当前单词 | S-F8   |
